<setAuthenticate>
- 로그인 상태를 관리하는 함수
- 로그인: true/ 로그아웃: false

<to>
- 리다이렉션(페이지간의 이동)을 위해 사용
- 로그인한 후에 사용자를 어디로 이동시킬지 결정하는 역할

<useSearchParams>
- URL의 검색 파라미터를 가져옴
- get메소드를 사용하면 특정 파라미터의 값을 가져올수 있음
- 특정 파라미터를 수정하거나 추가할수 있음

<useNavigate>
- 내비게이션 함수를 반환
- 다른 라우트로 이동하거나 뒤로 가거나 앞으로 가는 등의 작업을 수행

<Link>
- 웹사이트 내에서 새로운 페이지를 로딩하지 않고 다른 페이지로 이동할 수 있게 해주는 역할
